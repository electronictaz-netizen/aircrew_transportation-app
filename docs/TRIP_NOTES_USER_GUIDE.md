# Trip Notes & Comments User Guide

**Last Updated:** January 27, 2026

Trip Notes allow you to add comments, notes, and internal communications to trips. Perfect for sharing information with your team, documenting special instructions, and keeping a history of trip-related communications!

---

## What Are Trip Notes?

Trip Notes are comments and messages attached to trips. They help you:
- Share information with your team
- Document special instructions
- Keep a communication history
- Add private internal comments
- Mention team members

---

## Access Requirements

- **All Users**: Can view notes (except internal comments)
- **Managers**: Can add manager notes and internal comments
- **Drivers**: Can add driver notes
- Available on all subscription tiers

---

## Viewing Notes

### How to View Notes
1. Open any trip for editing
2. Click **"View Notes"** button
3. Notes panel expands showing all notes for this trip
4. Notes are displayed newest first (most recent at top)

### Note Display
Each note shows:
- **Author**: Name and role of who added the note
- **Type**: Manager Note, Driver Note, or Internal Comment
- **Timestamp**: When the note was created
- **Content**: The note text
- **Mentions**: Highlighted @username mentions
- **Internal Badge**: If it's an internal comment

---

## Adding Notes

### Step 1: Open Notes Panel
1. Edit any trip
2. Click **"View Notes"** button
3. Notes panel expands

### Step 2: Select Note Type
Choose from:
- **Manager Note**: Visible to all managers and admins
- **Driver Note**: Added by drivers about the trip
- **Internal Comment**: Private, manager-to-manager only

### Step 3: Write Your Note
1. Enter your note in the text area
2. Use **@username** to mention team members
3. For private notes, check **"Internal only"** checkbox
4. Click **"Add Note"**

### Note Types Explained

#### Manager Note
- **Who sees it**: All managers and administrators
- **Who can add**: Managers and administrators
- **Use for**: General trip information, instructions, updates

#### Driver Note
- **Who sees it**: All managers and administrators
- **Who can add**: Drivers
- **Use for**: Trip feedback, issues encountered, completion notes

#### Internal Comment
- **Who sees it**: Only managers and administrators (NOT drivers)
- **Who can add**: Managers and administrators
- **Use for**: Private discussions, sensitive information, internal notes

---

## Using @Mentions

### What Are @Mentions?
Tag team members in notes to notify them or draw their attention.

### How to Mention
1. Type **@** followed by username
2. Example: `@john` or `@sarah`
3. Mentioned users are highlighted in the note

### Best Practices
- ✅ Use @mentions for important updates
- ✅ Mention the driver for trip-specific questions
- ✅ Mention managers for approval requests
- ✅ Don't overuse - only mention when needed

**Note**: Currently, @mentions are visual only. Email notifications coming soon!

## Internal Comments

### What Are Internal Comments?
Private notes visible only to managers and administrators. Drivers cannot see these.

### When to Use
- Sensitive information about customers
- Internal discussions about pricing
- Manager-to-manager communications
- Private notes about trip issues

### How to Create
1. Select **"Internal Comment"** as note type, OR
2. Select **"Manager Note"** and check **"Internal only"**
3. Write your comment
4. Click **"Add Note"**

### Visual Indicator
Internal comments have:
- Blue border and background
- "Internal" badge
- Different styling to distinguish from regular notes

---

## Deleting Notes

### Who Can Delete
- Only the **author** can delete their own notes
- You cannot delete notes written by others

### How to Delete
1. Find your note in the notes list
2. Click **"Delete"** button on your note
3. Confirm deletion
4. Note is permanently removed

### Why Can't I Delete Someone Else's Note?
This prevents accidental deletion and maintains note history. If you need to remove someone else's note, contact an administrator.

- ✅ **Be Clear**: Write clear, concise notes

- ✅ **Be Timely**: Add notes when information is fresh
- ✅ **Use Types Correctly**: Choose the right note type

### Note Organization
- ✅ **Chronological**: Notes are sorted newest first
- ✅ **Use Mentions**: Tag relevant team members
- ✅ **Internal When Needed**: Use internal comments for sensitive info

### Communication Tips
- ✅ **Document Issues**: Add notes when problems occur

- ✅ **Special Instructions**: Document special requirements
- ✅ **Feedback**: Use driver notes for trip feedback

## Use Cases

### Customer Special Requests
**Scenario**: Customer requests specific route or stops
**Action**: Add Manager Note with details
**Result**: Driver sees note and follows instructions

### Trip Issues
**Scenario**: Driver encounters problem during trip
**Action**: Driver adds Driver Note explaining issue
**Result**: Managers see note and can follow up

### Internal Discussion
**Scenario**: Manager wants to discuss pricing with other managers
**Action**: Add Internal Comment
**Result**: Private discussion visible only to managers

### End-of-Trip Notes
**Scenario**: Driver completes trip and wants to share feedback
**Action**: Driver adds Driver Note
**Result**: Feedback recorded for future reference

## Troubleshooting

### Notes Not Appearing
- **Check**: Are you viewing the correct trip?
- **Check**: Refresh the page
- **Check**: Do you have permission to view notes?

### Can't Add Note
- **Check**: Are you logged in?
- **Check**: Do you have permission to add notes?
- **Check**: Is the notes panel open?
- **Try**: Refresh the page

### Can't Delete Note
- **Check**: Did you write this note?
- **Check**: Only authors can delete their own notes
- **Contact**: Administrator if you need a note removed

### Internal Comments Not Showing
- **Check**: Are you a manager or administrator?
- **Note**: Drivers cannot see internal comments
- **Check**: Is the note marked as internal?

## FAQ

**Q: Can I edit a note after adding it?**  
A: Currently, notes cannot be edited. Delete and re-add if needed.

**Q: How long are notes kept?**  
A: Notes are kept indefinitely with the trip.

**Q: Can I search notes?**  
A: Yes! Use the search bar in Trip Filters to search note content.

**Q: Do @mentions send notifications?**  
A: Currently visual only. Email notifications coming soon!

**Q: Can drivers see internal comments?**  
A: No, internal comments are only visible to managers and administrators.

**Q: How many notes can I add to a trip?**  
A: There's no limit! Add as many notes as needed.
