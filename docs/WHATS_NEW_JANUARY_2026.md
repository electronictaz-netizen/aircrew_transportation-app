#### Selecting Trips
1. In the Trip List, check the boxes next to trips you want to select

#### Bulk Status Update
1. Select one or more trips
2. Click **"Update Status"** button
3. Choose the new status from the dropdown
4. Click **"Update Status"** to confirm
5. All selected trips are updated instantly

#### Bulk Export to CSV
1. **Export Selected Trips:**
   - Select trips using checkboxes
   - Click **"Export (N)"** where N is the number selected
   - CSV file downloads with all trip details

2. **Export All Trips:**
   - Click **"Export All"** (no selection needed)
   - All trips are exported to CSV

**CSV Includes:**
- Trip ID, Category, Dates/Times
- Flight/Job Number
- Pickup/Dropoff Locations
- Passenger Count
- Customer, Driver, Status
- Actual Pickup/Dropoff Times
- Financial Data (Rates, Driver Pay)

### What It Does

### Key Benefits
- ğŸ“ **Better Communication**: Share information about trips with your team
- ğŸ”’ **Private Comments**: Internal notes visible only to managers
- ğŸ“š **History**: Full timeline of all notes and comments
- ğŸ‘¥ **Mentions**: Tag team members with @username

### How to Use

#### Viewing Notes
1. Open any trip for editing
2. Click **"View Notes"** button
3. Notes panel expands showing all notes for this trip

#### Adding a Note
1. In the Notes panel, select note type:
   - **Manager Note**: Visible to all managers
   - **Driver Note**: Added by driver after trip
   - **Internal Comment**: Private, manager-to-manager only
2. Enter your note in the text area
3. Use **@username** to mention team members
4. Check **"Internal only"** for private comments
5. Click **"Add Note"**

#### Note Types Explained
- **Manager Note**: General notes visible to all managers and admins
- **Driver Note**: Notes added by drivers about the trip
- **Internal Comment**: Private notes only visible to managers/admins (not drivers)

#### Deleting Notes
- Only the author can delete their own notes
- Click **"Delete"** button on your note
- Confirm deletion

### Access Requirements
- **All Users**: Can view notes (except internal comments - managers only)
- **Managers**: Can add manager and internal notes
- **Drivers**: Can add driver notes
