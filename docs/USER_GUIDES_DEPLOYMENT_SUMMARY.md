## üöÄ Deployment Instructions

### For User Guides

#### Option 1: In-App Documentation
1. Convert markdown files to HTML (if needed)
2. Upload to `/public/docs/` folder in the app
3. Add links in the app's Help section
4. Users can access via: `https://your-app.com/docs/`

#### Option 2: Marketing Website
1. Convert markdown to HTML
2. Upload to marketing website `/docs/` folder
3. Link from main navigation
4. Users can access via: `https://tazsoftware.biz/docs/`

#### Option 3: External Documentation Site
1. Upload markdown files to GitHub Pages, GitBook, or similar
2. Link from app and marketing website
3. Keep documentation separate from app code

### For Marketing Website

#### Upload to GoDaddy (or your hosting)
1. **Upload new files:**
   - `features.html`
