# Manager Quick Reference Card

A one-page quick reference for managers using the Aircrew Transportation app.

## Logging In

1. Go to app URL
2. Enter email and password
3. Click "Sign In"

**Forgot Password?** Click "Forgot Password" and check your email.

---

## Creating a Trip

1. Click **"Create Trip"**
2. **Select Airport** (choose from your company's configured airports)
3. **Enter Flight Number** (e.g., UA1234)
4. **Set Pickup Date & Time**
5. **Choose Pickup Location**:
   - Use Airport
   - Use Saved Location
   - Enter Text (custom)
6. **Choose Dropoff Location** (same options)
7. **Enter Passenger Info** (name, contact, instructions)
8. **Check "Recurring"** if needed
9. Click **"Save Trip"**

---

## Managing Drivers

**Access**: Click "Manage Drivers"

**Add Driver**:
- Name, Email, Phone (required)
- License Number
- Notification Preference (Email, SMS, or Both)
- Set Active status

**Edit Driver**: Click "Edit" → Update → Save

**Deactivate**: Edit → Uncheck "Active" → Save

---

## Managing Locations

**Access**: Click "Manage Locations"

**Add Location**:
- Name (descriptive)
- Full Address
- Description (optional)
- Set Active

**Use in Trips**: Select "Use Saved Location" when creating trips

---

## Assigning Trips

### Single Assignment
1. Find trip in list
2. Click **"Assign"**
3. Select driver
4. Click **"Assign"**

### Bulk Assignment
1. **Check boxes** next to multiple trips
2. Click **"Assign"** button
3. Select driver
4. Click **"Assign"**

**Reassign**: Same process, select different driver

---

## Daily Assignment Emails

1. Click **"Send Daily Assignment Emails"**
2. Choose: Email, SMS, or Both
3. Click **"Send"**

**When**: Evening (for tomorrow's trips)

**What Drivers Get**: List of trips, times, locations

---

## Filtering Trips

**By Airport**: Click airport filter buttons (your company's airports will be listed)

**By Date**: 
- Today, Tomorrow, This Week, Next Week
- Custom Range

**By Driver**: Select from dropdown

**By Status**: Assigned or Unassigned

**Auto-sorted** by pickup date when filters change

---

## Editing/Deleting Trips

**Edit**: Click "Edit" → Update → Save

**Delete**: Click "Delete" → Confirm

**Bulk Delete**: Check trips → Click "Delete" → Confirm

---

## Airport Information

The airports available in your system are configured by your company administrator. Airport names and codes will be displayed in trip forms and lists.

---

## Notification Preferences

**Both**: Email + SMS  
**Email Only**: Email only  
**SMS Only**: Text only (requires phone number)

**Set in**: Manage Drivers → Edit Driver

---

## Recurring Trips

1. Create trip normally
2. Check **"Recurring Trip"**
3. Select frequency: Daily, Weekly, Monthly
4. Set end date (optional)
5. Save

**System generates** upcoming trips automatically

---

## Best Practices

✅ Create trips 24+ hours in advance  
✅ Use saved locations for speed  
✅ Assign trips promptly  
✅ Send daily summaries in evening  
✅ Keep driver info updated  
✅ Use bulk assignment for efficiency  
✅ Review unassigned trips daily  

---

## Common Tasks

**Morning Routine**:
- Review today's trips
- Check unassigned trips
- Assign any pending trips

**Creating Trips**:
- Use saved locations
- Include passenger contact
- Add special instructions

**Evening Routine**:
- Review tomorrow's trips
- Ensure all assigned
- Send daily summaries

---

## Troubleshooting

**Trip won't save**: Check required fields  
**Driver not notified**: Verify email/phone and preferences  
**Location missing**: Check if location is active  
**Can't assign**: Verify drivers are active  
**Emails not sending**: Check trips exist for tomorrow
